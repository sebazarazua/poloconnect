# Assets

## Logo

Coloca tu imagen del logo como `logo.png` en esta carpeta.

**Especificaciones recomendadas:**
- **Nombre del archivo:** `logo.png`
- **Tamaño:** 120x40 píxeles (o con esa proporción)
- **Formato:** PNG con fondo transparente (recomendado)
- **Contenido:** Tu logo de Polo Connect

Una vez colocada la imagen, aparecerá automáticamente en la barra superior de la app.
